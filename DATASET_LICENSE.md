# OLSD Dataset License Policy

The OLSD codebase is licensed under **Apache-2.0**.

Individual datasets within OLSD carry their own licenses as specified in each
episode's metadata. All contributed data must use one of the following
permissive licenses:

- **CC0 1.0 Universal** (Public Domain Dedication) — preferred
- **CC-BY 4.0** (Creative Commons Attribution 4.0 International)
- **MIT License**

Proprietary, non-commercial, or otherwise restricted data will not be
included in the public OLSD dataset.

## Attribution

When using OLSD data, please cite the original contributors as specified
in each episode's `metadata.contributor` and `metadata.citation` fields.

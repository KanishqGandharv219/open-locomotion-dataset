Quick Start

Get running in three commands.

terminal
# Install pip install -e . # Generate expert trajectories olsd generate -r halfcheetah -p checkpoints/halfcheetah/ppo/best/best_model.zip -n 500 # Train a PPO baseline olsd train -r walker2d -a sac -t 1000000 # Evaluate all trained models olsd eval --all -n 50
